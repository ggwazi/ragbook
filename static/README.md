# Static Assets

This directory contains static assets served by the Ragbook server.

## Usage

Place static files (CSS, JavaScript, images, etc.) in this directory.
They will be available at `/static/{filename}`.

## Example

```
static/
├── css/
│   └── style.css
├── js/
│   └── app.js
└── images/
    └── logo.png
```

Access them at:
- `/static/css/style.css`
- `/static/js/app.js`
- `/static/images/logo.png`
